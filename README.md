# Ada Icons

A modern and mature icon pack inspired by copilot with a focus on data and AI.


## Installation

You can install the icon pack via npm:

```bash
npm install adaicons
```

# Usage

```
javascript
import { Icon1, Icon2 } from 'your-icon-pack-name';

// Usage in JSX (for React):
<Icon1 />
```
